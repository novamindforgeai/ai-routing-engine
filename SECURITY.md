# Security Policy

This document outlines security expectations and responsible disclosure
for the AI Routing Engine.

---

## Scope

This repository contains proprietary infrastructure code used for
AI routing, decision-making, and traffic orchestration.

It is **not intended for public deployment** without an Enterprise License.

---

## Supported Versions

Only versions released under an explicit Enterprise License
are considered supported for production use.

Public repository versions are provided **for evaluation only**.

---

---

## Disclosure Policy

- Do NOT publicly disclose vulnerabilities
- Do NOT open public issues for security findings
- Do NOT exploit vulnerabilities in production systems

We commit to reviewing reports and responding in good faith.

---

## No Bug Bounty

At this time, there is no public bug bounty program.
Security reports are handled on a case-by-case basis.

---

## Disclaimer

Unauthorized testing, scanning, or exploitation of this system
without permission is prohibited.

Use of this software is subject to licensing terms.
